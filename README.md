# Scouting Hexapod – 3D CAD Models

An interactive 3D showcase of all CAD parts used for the Scouting Hexapod robot.

## 🌐 Live Viewer

**[View the 3D models → `https://truli134.github.io/Scouting-Hexapod-3D-CAD-Models`](https://truli134.github.io/Scouting-Hexapod-3D-CAD-Models)**

The page uses the GitHub API to automatically discover every `.stl` file in this repo and render it interactively in the browser — no downloads or software required.

---

## 🚀 Enabling GitHub Pages (one-time setup)

1. Go to your repo on GitHub → **Settings** → **Pages**
2. Under **Source**, select **Deploy from a branch**
3. Set the branch to **`main`** and the folder to **`/ (root)`**
4. Click **Save**

GitHub will publish the site within ~30 seconds. The URL will be:
`https://truli134.github.io/Scouting-Hexapod-3D-CAD-Models`

---

## 📁 Adding STL Files

Just commit any `.stl` files (in any folder structure you like) and they will appear in the viewer automatically — no code changes needed.
